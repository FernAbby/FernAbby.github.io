CSS3弹性布局

发表于 2017-05-17 | 分类于 css | | 阅读次数

弹性盒布局模型（Flexible Box Layout）是 CSS3 规范中提出的一种新的布局方式。该布局模型的目的是提供一种更加高效的方式来对容器中的条目进行布局、对齐和分配空间。这种布局方式在条目尺寸未知或动态时也能工作。这种布局方式已经被主流浏览器所支持，可以在 Web 应用开发中使用。

容器设置

flex-flow是flex-direction与flex-wrap属性的简写

flex-direction:弹性项目在容器中的放置方向，row(行内方向，从左到右)
flex-wrap: 是否需要拆行




容器中条目设置

order: -1;把条目提到第一位

flex-grow:1;当前容器有多余空间时，这些条目按不同的比例分配

flex-shrink:当前容器空间不足时，各个条目按相应比例缩小
flex-basis:接受值与width属性相同，用于确定弹性条目的初始主轴尺寸

条目对齐

自动空白边（作用在条目）

第一种方式是使用自动空白边，即”margin: auto”。在使用自动空白边时，容器中额外的空白空间会被声明为 auto 的空白边占据。

主轴方向对齐（作用在容器）

在主轴方向上的对齐。这是通过容器上的”justify-content”属性来进行设置，可以调整条目在主轴方向上的对齐方式。

交叉轴方向对齐（作用在容器）

方式是交叉轴方向上的对齐。除了在主轴方向上对齐之外，条目也可以在交叉轴方向上对齐。容器上的属性”align-items”用来设置容器中所有条目在交叉轴上的默认对齐方向，而条目上的属性”align-self”用来覆写容器指定的对齐方式。

交叉轴空白处理

当容器在交叉轴方向上有空白空间时，属性”align-content”用来对齐容器中的行。该属性的作用类似于”justify-content”，只不过”justify-content”是在主轴方向上对齐行中的条目。当容器中只有单行时，该属性不起作用。

浏览器支持

对于这些浏览器相关的前缀，最好使用 autoprefixer 这样的工具来进行处理。
