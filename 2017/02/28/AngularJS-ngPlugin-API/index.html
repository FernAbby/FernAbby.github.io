AngularJS ngPlugin API

发表于 2017-02-28 | 分类于 BMS-UI | | 阅读次数

关于用户系统／业务系统里常用到的AngularJS自定义服务、指令和过滤器的简单API.重点在于服务和指令，过滤器列了三个实例，其他的不做枚举。调用示例只做参考，具体问题具体分析，在开发过程中对公用插件库的修改，API会动态更新。当前版本：ngPlugin-1.1.0.js

服务的引用都需要注入

alert服务

alert

  • message: 提示信息
  • reason: 提示信息补充说明
  • type: fail|success
  • delay: 显示时间，单位－秒

confirm

  • title: 提示标题
  • detailed: 提示详情信息
  • no: 取消按钮文案
  • yes: 确认按钮文案
  • delay: 显示时间，单位－秒
  • confirm: 点击确认后执行的操作
  • cancel：点击取消按钮后执行的操作

ngSubmit服务

  • ele: 点击触发提交的元素
  • data: 提交数据，此服务为post提交，默认数据格式－json
  • form: 以表单名字命名的局部作用域变量，做表单验证要用到
  • success: 提交后code==0时执行函数
  • failed: 提交后code!=0执行函数
  • error: 网络异常时执行函数
  • beforePost：数据提交前执行的操作

stModal服务

  • templateUrl: 模版引用链接
  • controller: 模版对应的controller
  • resolve: 传递到模态框的参数－对象
  • ok: 模态框确认按钮执行函数
  • cancel: 模态框确认按钮执行函数

stDateFormat服务

date

获得YY-MM-DD格式日期

time

获得回距1970年1月1日之间的毫秒数

stAutoValidate

  • form: 以表单名字命名的局部作用域变量
  • success: 验证成功后执行函数
  • fail: 验证失败后执行函数
  • isAlert: 验证失败后是否提示。默认true

stValidate指令

  • name(dy-name)： 唯一性，必写项
  • trigger： 触发验证的事件
  • validateMax(Msg)： 最大值验证（<=）
  • validateMin(Msg)： 最小值验证（>=）
  • validateNotBlankMsg： 不能为空验证
  • validateMobileMsg： 手机号码验证()
  • validateTelMsg： 电话验证
  • validateIdNoMsg： 身份证验证
  • validateEmailMsg： 邮箱验证
  • validateAlphabetMsg： 数字字母验证
  • validateLenMax(Msg)： 最大长度验证（<=）
  • validateLenMin(Msg)： 最小长度验证（>=）
  • validateLicenseMsg： 车牌号验证
  • validatePast(Msg)： 是否过时验证
  • validateRepeat(Msg)： 是否一致验证，如密码确认
  • validateNumberMsg： 数字验证
  • validateIntegerMsg： 整数验证
  • validateNaturalNumberMsg： 自然数验证
  • validateDecimal(Msg)： 小数位数验证
  • validateUrlMsg： Url验证
  • validatePasswordMsg： 密码验证

stDropdown指令

  • n：定义下拉框显示行数
  • width：定义下拉框宽度
  • disabled： 下拉框不可用
  • options：option初始值（可选）
  • label： 显示属性定义（可选）
  • value： 值属性定义（可选）

stAutoComplete指令

  • n：定义备选显示行数
  • array：全部备选数据
  • query：筛选条件实际为ng-model（必写）
  • idValue：需要的数据值（针对数据为对象时）
  • idName：字段名称（针对数据为对象时）

stGeocode | stBGeocoder指令

引入百度地图

  • mapaddress： 详细地址
  • longitude： 经度
  • latitude： 纬度

引入高德地图

  • mapaddress： 详细地址
  • longitude： 经度
  • latitude： 纬度

stTooltip指令

  • li： 提示内容
  • name： 属性名
  • trigger：触发事件 focus|mouseenter

stSeatTooltip指令

  • li： 提示内容
  • name： 属性名
  • trigger：触发事件 focus|mouseenter

stModalScroll指令

  • array： 请求响应返回数据数组
  • data： 请求数据
  • method：请求方式 get|post
  • url： 请求链接

stTimepicker指令

  • bootstrap的timepicker插件

stDatetimepicker指令

  • bootstrap的Datetimepicker插件

stTagsortable指令

  • tagsarray: tag数组

stSeat指令

  • data： 座位数据
  • col: 多少列
  • row： 多少行
  • number： 座位数

stRadio指令

value为字符串

value为对象

  • ng-checked:需要设置初始值时
  • ng-value: 需要绑定的数据

stCheckbox指令

  • disabled: 复选框禁用

stUeditor指令

  • config: normal | upload 分为可上传图片和不可上传图片
  • url: 图片上传链接
  • setdisabled: 不可编辑

stEditor指令

  • server: 图片上传链接

stTextareaValidate指令

  • validateLenMax(Msg)： 最大长度验证（<=）
  • validateLenMin(Msg)： 最小长度验证（>=）
  • validateNotBlankMsg： 不能为空验证

stPopover指令

  • describe: 数据表头
  • datas: 数据表
  • popover-placement：popover显示位置
  • popover-title：popover标题
